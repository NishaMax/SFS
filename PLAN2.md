4. **Gallery (`/gallery`)**
   - A masonry layout showing high-quality photos of the physical store, happy customers, and product shelves.

5. **Promotions (`/promotions`)**
   - Current offers and seasonal sales banners.

6. **Contact & Location (`/contact`, `/location`)**

7. **Legal Pages (`/faq`, `/privacy`, `/terms`)**
   - Simple, clean typography-focused pages for store policies.

## Phase 4: Backend & Admin Dashboard Integration (Future 🚀)

Once all UI is complete and perfect, we will move to make the website dynamic.

1. **Database Setup (Firebase / Supabase)**
   - Create schemas for Products, Categories, and Promotions.
   
2. **Media Storage (Cloudinary / Firebase Storage)**
   - Host all product images and gallery photos optimally.
   
3. **Admin Dashboard (Owner Portal)**
   - Secure login for store owners.
   - Interface to **Add / Edit / Delete** products and update prices.
   - Ability to update stock status (In Stock / Out of Stock).
   
4. **API Integration**
   - Connect the frontend pages (Categories, Featured Products, New Arrivals) to fetch data live from the database instead of using hardcoded mock data.
